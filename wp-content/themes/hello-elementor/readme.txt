This is a child theme of Hello Elementor.  
Use this theme if you want to customize the Hello Elementor theme safely.

== Description ==  
Hello Elementor Child Theme is a lightweight and fast WordPress theme that inherits the features and design of the parent "Hello Elementor" theme. It allows you to make customizations without affecting the original theme files, so updates to the parent theme won't overwrite your changes.

== Installation ==  
1. Upload the `hello-elementor-child` folder to the `/wp-content/themes/` directory.  
2. Activate the theme through the 'Appearance > Themes' menu in WordPress.  
3. Make sure the parent "Hello Elementor" theme is also installed.  

== Customization ==  
You can add your custom CSS in the `style.css` file and additional functions in `functions.php`.
